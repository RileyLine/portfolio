MAPS

Ancient

On May 3rd, 2021 when Operation Broken Fang concluded, Ancient replaced Train in the Active Duty Map pool while Train went into the Reserve Map pool.

Created By: Valve
First Released: DEC 2020
Location: Unknown
Theme: Rainforest

Dust II

Dust II is arguably the most popular and played Counter-Strike map and perhaps the iconic map of the franchise.

Created By: Dave Johnston
First Released: MAR 2001
Location: Morocco
Theme: Desert

Inferno

The map is set in a small town with European architecture. The presence of the Separatist faction suggests that the map is definitely set in Italy, as evidenced by the signs written in Italian seen around the map.

Created By: "Narby"
First Released: MAR 2001
Location: Italy
Theme: Urban

Mirage

The map is themed like a Middle Eastern town, but because of its close resemblance to Moroccan style architecture, it is likely set somewhere in Morocco.

Created By: Michael Hull
First Released: 2004 (de_cpl_strike)
Location: Morocco
Theme: Suburban

Nuke

The map takes place in a warehouse containing nuclear materials or a nuclear power plant. The original version of the map takes place in a German nuclear power plant, while the revamped version takes place in an American one.

Created By: Valve
First Released: MAY 1999
Location: USA
Theme: Industrial

Overpass

Overpass is a bomb defusal map that was introduced in the Winter Offensive update along with Cobblestone. It was CS:GO's first completely new defuse map that was designed specifically for use in competitive play.

Created By: Valve
First Released: DEC 2013
Location: Berlin
Theme: Urban

Vertigo

Vertigo is based on a skyscraper that was under construction and centered the conflict between the Counter-Terrorist and the Terrorist teams.

Created By: "Narby"
First Released: MAR 2001
Location: USA
Theme: Urban
